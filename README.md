# Análisis de Instituciones Bancarias FDIC (Solemne 1)

**Autor:** Benjamín González Alfaro  
**Carrera:** Ingeniería Física, UNAB.

## Descripción
Este proyecto realiza una minería de datos sobre el dataset público de la FDIC en BigQuery. Se analizan activos y depósitos bancarios desde 1970 hasta 2019.

## Hallazgos Clave
- Se utilizó **Escala Logarítmica** para comparar estados con alta disparidad financiera (NY vs. estados rurales).
- El análisis segmentado muestra una consolidación de capital al rededor del año 2008.
